## 프로젝트 설명

React 공부 및 학교 시험 공부를 할 때 암기 과목들을 보다 쉽게 공부 하기 위해 만든 프로그램 입니다.
<br/>
미완성 프로젝트이지만 유용하게 사용 하고 있습니다.
